+ ### 1.先打开官网下载typora，typora官网地址：https://typoraio.cn
{% asset_img 1.jpg %}
找到历史版本，下载1.6.7版本
{% asset_img 2.jpg %}

+ ### 2.双击打开typora安装包，选择install for all users（recommended）
{% asset_img 3.jpg %}

+ ### 3.我这里选择的是E盘,记得此目录
{% asset_img 4.jpg %}

+ ### 4.勾选，然后next，然后记得退出typora，typora打开是无法激活成功的
{% asset_img 5.jpg %}

+ ### 5.下载激活文件
[点击下载](winmm.dll "激活文件")

+ ### 6.将下载好的文件，拖动到刚刚装好的typora目录下
{% asset_img 8.jpg %}

+ ### 7.打开typora，点击帮助》》》我的许可证
{% asset_img 6.jpg %}

+ ### 8.显示已激活成功
{% asset_img 7.jpg %}
