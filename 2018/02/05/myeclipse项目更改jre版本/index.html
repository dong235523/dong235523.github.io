myeclipse项目更改jre版本

发表于2018-02-05|更新于2022-11-22|后端技术
|阅读量:

1.先安装jre
2.项目右键Properties
3.点击Java Build Path
4.在Libraries选项选择Add Library..
5.next
6.选择下边选项框Installed JREs
7.点击add
8.JRE Type界面选择Standard VM 点击next
9.找到jre安装目录
10.在下边勾选相应的jre版本
