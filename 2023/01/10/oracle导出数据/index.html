oracle导出数据

发表于2023-01-10|更新于2023-01-10|数据库
|阅读量:

  • 创建表空间

    • 使用Enterprise Manager Console

    • 账号密码都是sys，连接身份选择sysdba

    • 创建表空间，选项都是默认即可

    • 右键创建好的表空间，点击 添加数据文件

    • 这个一般信息不用管

    • 存储配置含义，数据存满后每次扩展3M，上限3个G

  • 创建用户

    • 一般信息页面中输入用户名密码，选择表空间

    • 角色中添加DBA，勾选上管理选项，点击创建即可

  • 导出数据

    • 先导出一个建表语句sql

    • 导出数据最好用plsql的PDE格式，兼容性比较好

    • Tools ->Export Tables

    • 然后务必记住还要导出数据库的序列等等

    • Tools ->Export User Objects

    • 把一些序列、视图等信息导出。
